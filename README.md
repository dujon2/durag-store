# Durag Store

A full-stack e-commerce portfolio project for selling durags.

## Tech Stack

- React
- Vite
- Python
- FastAPI
- REST API
- CSS

## Features

- Product listing
- Product cards
- Add to cart
- Cart total calculation
- Python backend API
- React frontend connected to backend

## How to Run

### Backend

```bash
cd backend
venv\Scripts\activate
uvicorn main:app --reload
